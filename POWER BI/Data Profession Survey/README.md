#🎲 **Data  Profession Survey**

#📌**Overview**
This project was done to display how various data professions were faring in a certain  geographical region. An interactive dashboard was built elaborating role preferences count, employee count by the geographical region, employee work satisfaction and gender distribution.
This analysis can be ideal in elaborating how data profession is impacting the world and aid in formulating future plans in   data careers.

#🎯**Objectives**
-Display various data roles.
-	Display data roles count.
-	Calculate average age in each data role.
-	Show gender distribution in each data role.
-	Show  data roles count in the geographical count.

#📊**Datasets**
Source ;Kaggle
#📈**Key Performance Indicators**
-	Role Preferences
-	Employee Count
-	Average Age
-	Employee Work satisfaction count
-	Employee Gender Distribution

#🌟**Insights**
Data Analyst was the role with highest preference of the total employee population occupying 381  out of 630 count making 60.47% of the entire population.44.4% of male in the whole population  opted for this role while only  16% female selected this role.
The learning personnel were 90 out of 630 count making 14% of the entire population.26.35% were undecided on whether the data roles were easy or hard.

#✍**Recommendation.**
Since Data profession is growing at an alarming rate ,More data analysts  roles vacancies should be created in order to accommodate the data analysts professionals who are in the job market.
